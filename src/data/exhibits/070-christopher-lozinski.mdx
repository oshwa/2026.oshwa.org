title: Wiki about About Source Video Processing
---

This is a wiki about processing video in real time using microcontrollers, ASICs and FPGAs. It is slowly expanding to cover other Open Source topics.  The problem is that there is no good directory of Open Source solutions.  The information is scattered all over the internet.  Worse yet, the search engines prioritize closed source advertising-driving products.  Here required commercial solutions are listed last.  And even worse is that the search engines are using climate destroying AI.  Here the content is all human curated.
